# Highlights
## ดูข้อมูลแหล่งที่มาจากโฆษณาในหน้าบทสนทนา
![ดูข้อมูลแหล่งที่มาจากโฆษณาในหน้าบทสนทนา](https://public.kaojai.ai/changelog/attachments/2026-04-21/a56e04a8-95ac-4803-bcc9-b65c21abbdf1-a9b699369fd1.jpg)


ตอนนี้คุณสามารถดูข้อมูลประกอบว่าแชทมาจากที่ไหน (เช่น มาจากโฆษณา) ได้โดยตรงในไทม์ไลน์บทสนทนา เพื่อช่วยติดตามและตอบกลับได้เร็วขึ้น


# ฟีเจอร์
- **สร้างใบแจ้งหนี้อัตโนมัติทุกวัน**
  - ระบบจะสร้างใบแจ้งหนี้ให้อัตโนมัติทุกวันเมื่อรอบบิลของแพ็กเกจแบบชำระเงินสิ้นสุดลง
  - ช่วยลดงานทำมือและทำให้ข้อมูลการเรียกเก็บเงินอัปเดตเสมอ
  - เพิ่มความถูกต้องและลดโอกาสตกหล่นของใบแจ้งหนี้
  - คีย์เวิร์ด: สร้างใบแจ้งหนี้, ระบบบิลอัตโนมัติ, ใบแจ้งหนี้รายวัน, แพ็กเกจแบบชำระเงิน.
- **ปรับดีไซน์หน้าแอดมินให้เรียบง่ายขึ้น**
  - ปรับเลย์เอาต์หน้าแอดมินใหม่ให้ดูสะอาดตาและโฟกัสกับงานมากขึ้น
  - นำส่วนหัวออกเพื่อลดความรกของหน้าจอ
  - การจัดวางเมนูและระยะห่างของเนื้อหาดูง่ายขึ้น
  - คีย์เวิร์ด: ปรับหน้าแอดมิน, เลย์เอาต์ใหม่, การใช้งานสะดวกขึ้น.

# แก้ไขบั๊ก
- **ปรับปรุงความเสถียรของการเข้าสู่ระบบด้วย LINE** — แก้ไขปัญหาที่อาจทำให้ผู้ใช้บางรายไม่สามารถเข้าสู่ระบบด้วย LINE ได้

## See ad referral context in chat
![See ad referral context in chat](https://public.kaojai.ai/changelog/attachments/2026-04-21/a56e04a8-95ac-4803-bcc9-b65c21abbdf1-a9b699369fd1.jpg)


You can now see helpful context about where a chat came from (such as an ad referral) directly in the chat timeline for quicker follow-ups.


# Features
- **Automatic daily invoice generation**
  - Invoices are now generated automatically each day when a billing period ends for paid plans.
  - This helps keep billing up to date without manual steps.
  - Improves accuracy and reduces missed invoices.
  - Keywords: invoice generation, billing automation, daily invoices, paid plans.
- **Cleaner admin layout**
  - The admin page has a refreshed layout for a cleaner, more focused workspace.
  - The header area has been removed to reduce clutter.
  - Navigation and content spacing are now easier to scan.
  - Keywords: admin redesign, clean layout, improved navigation.

# Bug Fixed
- **Improved LINE login reliability** — Fixed an issue that could prevent some users from signing in with LINE.
