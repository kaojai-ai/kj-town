# ฟีเจอร์
- **ใช้ประวัติแชตมากขึ้นเพื่อเข้าใจบริบทได้ดีขึ้น**
  - นำประวัติการสนทนาล่าสุดมาใช้เป็นบริบทมากขึ้นเวลาสร้างคำตอบ
  - ช่วยให้คำตอบต่อเนื่องและสอดคล้องกัน ลดการต้องพิมพ์ซ้ำ
  - เหมาะกับเคสที่คุยยาวหรือมีการติดตามต่อเนื่องในชีวิตประจำวัน
  - คีย์เวิร์ด: บริบทการสนทนา, ประวัติแชต, คำตอบต่อเนื่อง, งานซัพพอร์ตลูกค้า.
- **กรองเคสได้เร็วขึ้น และมุมมองติดตามชัดเจนขึ้น**
  - เพิ่มตัวกรองแบบเลือกได้เร็วขึ้น และเลือกช่วงวันที่ได้ง่ายกว่าเดิม เพื่อค้นหาเคสตามช่วงเวลาและประเภท
  - แยกเคสทดสอบออกจากงานจริง เพื่อไม่ให้ปะปนกับข้อมูลและตัวชี้วัด
  - ทำให้เห็นยอดรวมสำคัญชัดขึ้น ช่วยติดตามภาพรวมได้ในพริบตา
  - คีย์เวิร์ด: ตัวกรองเคส, ติดตามเคส, กรองตามวันที่, ตัวชี้วัดงานซัพพอร์ต.

# แก้ไขบั๊ก
- **แก้ปัญหารายการบทสนทนาล่าสุดซ้ำ** — รายการบทสนทนาล่าสุดจะไม่แสดงซ้ำอีกต่อไป
- **ปรับการทำงานของร่างข้อความให้สะอาดขึ้นเมื่อปิดสมาร์ทรีพลาย** — เมื่อปิดหน้าสมาร์ทรีพลาย ระบบจะล้างร่างข้อความและการทำงานที่เกี่ยวข้องได้ถูกต้อง
- **รายละเอียดเคสแสดงถูกต้องในข้อความที่แชร์** — รายละเอียดเคสจะแสดงได้ถูกต้องเมื่อแชร์ในรูปแบบข้อความ

# Features
- **More chat history used for better context**
  - Uses a larger amount of recent conversation history when generating responses.
  - Helps keep replies consistent and reduces the need to repeat information.
  - Improves day-to-day handling of longer, ongoing customer conversations.
  - Keywords: conversation context, chat history, consistent replies, customer support.
- **Faster filtering and clearer tracking views**
  - Added quicker filters and easier date selection to find cases by time period and type.
  - Separated test cases so they don’t mix with real work and metrics.
  - Improved visibility of key totals to help teams track progress at a glance.
  - Keywords: issue filtering, case tracking, date filters, support metrics.

# Bug Fixed
- **Fixed duplicate conversations in the recent list** — Recent conversations no longer appear as duplicate entries.
- **Cleaner draft behavior when closing smart replies** — Draft and related actions now clear correctly when you close the smart reply view.
- **Issue details now show correctly in shared messages** — Issue details now display properly when shared in message format.
